1. ARAD 앱은 유저의 참여에 대한 보상을 제공하는 앱이다
2. 사용자 위치, 카메라, 알림 등의 기능을 바탕으로 구성됨
3. 업체는 광고 시청을 통해 마케팅 효과를 가져온다

# 3) 설명

## 서버 송수신

서버와 http 통신을 하여 실시간으로 데이터를 공유\
앱과 서버간의 REST API 통신을 도와주는 라이브러리로 Retrofit2를 사용함

<img src="https://github.com/rohhyungwoo/ARAD_Public/assets/67363759/30422454-1f48-4966-8476-889389b94ca5" width="50%" height="100%"/>

유저의 소셜 정보, AR 이미지, 위치 정보 등을 서버에 업로드 한다

<img src="https://github.com/rohhyungwoo/ARAD_Public/assets/67363759/85e0b40c-5a22-40f2-86ad-37c5e682f1aa" width="50%" height="100%"/>

## AR 컨탠츠 실행
보다 입체적인 AR 이미지를 표현하기 위해 Unity를 통해 개발된 프로젝트를\
기존 안드로이드 앱에 적용하는 Uaal(Unity as a Library)을 사용
